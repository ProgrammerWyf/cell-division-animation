# cell-division-animation
一个基于PyGame的展示细胞分裂过程染色体等的变化的程序
## 进程
- 2025.3.29 立项
- 2025.4.27 v1.0 完成
- 2025.5.14 调整完成
- 2025.5.15 展示

## 功能
展示植物细胞有丝分裂
**注意！由Python2语法写成，在Python3运行时请留意！**

## 依赖及其他
- Pygame
 `pip install pygame`
- Py2exe
`pip install py2exe_py2`
- 编译为exe
在项目目录下运行cmd `python setup.py py2exe`
